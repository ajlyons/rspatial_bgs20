Mapping the Coronavirus

Overview

The Data

The New York Times compiles data on Coronavirus infections and deaths by county for the entire USA, drawing upon a variety of sources. They share these data on GitHub.

In this exercise, we will map the outbreak by county, using functions from sf and dplyr.

Workflow Overview

  1. Get the prevalance data from the New York Times GitHub site
  2. Filter the data to the most recent date
  3. Download the county boundaries for the entire USA as a Simple Features Data Frame
  4. Join the tables
  5. Create a choropleth map

About FIPS Codes

‘fips’ stands for Federal Information Processing Standard Publication 6

Many USA datasets identify counties by a 5-digit FIPS County Code. The first two digits identify the state, while the last three identify the county.

The tigris package includes the entire list as a data frame (fips_codes), and a lookup function:

tigris::lookup_code("MD", "Montgomery")
## [1] "The code for Maryland is '24' and the code for Montgomery County is '031'."

3. Download the county boundaries for the entire USA

There are a handful of R packages that ‘bundle’ administrative boundaries, and/or provide functions you can use to download them (e.g., USABoundaries, raster). We’ll use the tigris package, which lets you download a variety of administrative boundaries from the US Census Bureau.

## Download County Boundaries for the Whole USA
library(tigris)
options(tigris_use_cache = TRUE)   ## Tell tigris to use cached data if available

## Create a vector that contains the names of the lower 48 states
## 'state.name' is a built-in constant that contains the names of all 50 states
lower48 <- state.name[ -c(2,11)]     ## note the use of negative indices, which returns everything *except*
lower48                              ## those elements (in this case Alaska and Hawaii)

## Grab the '1:5million' resolution county boundaries for the lower 48 states
usa_counties_5m <- tigris::counties(state = lower48,      ## the name(s) of the states to get
                                    cb = TRUE,            ## use 'cartographic boundaries' data
                                    resolution = "5m",    ## which resolution of data to use
                                    class = "sf")         ## give us a sf object (as opposed to sp)

## Plot the boundaries
plot(usa_counties_5m %>% st_geometry())

5. Create a choropleth map

Finally, the fun part!

## Create a choropleth map of the number of COVID-19 cases per county
usa_map <- tm_shape(usa_cnty_covid19) + 
  tm_fill (col = "cases",                               ## the column that contains the values
           n = 6,                                       ## number of 'bins' to create
           style = "pretty",                            ## method to compute the 'bins
           palette = "YlOrRd",                          ## a nammed palette or a vector of colors
           colorNA = "grey50",                          ## the color to use for NA values (default is transparent)
           legend.reverse = TRUE ,                      ## reverse the legend?
           title = "cases"                              ## title for the legend element
  ) + tm_layout(legend.position = c("right", "bottom")) ## Where to put the legend
