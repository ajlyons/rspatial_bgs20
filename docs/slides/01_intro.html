Advice for Beginners

  1. Try to get working code.
  2. Focus on adapting code rather than starting from scratch.
  3. Pay attention to syntax.
  4. Pay attention to object types.
  5. Use meaningful variable names.
  6. Ask for help for cryptic error messages.
  7. Don’t aim for 100% understanding all at once.

R & RStudio

  • a programming language to write scripts
  • software (“environment”) that interprets scripts written in R
  • inspired by ‘S’ (1976), first beta version ~2000
  • many user-contributed packages: CRAN: >17,000; Bioconductor: >1,900; R-Forge: >2,100; GitHub: >57,000 [source]

  • a very popular tool (IDE) to work with both R software and R scripts
  • Version 1.0 came out in late 2016
  • RStudio needs R to run, but you can use R without RStudio
  • NEW: RStudio Cloud in beta - run RStudio through a browser

R. vs. Desktop GIS

  • familiar
  • reproducible analysis
  • batch runs
  • integration with other analyses
  • strong geospatial support - data & analyses
  • development of new methods
  • free and open source
  • one-off maps
  • serious cartography
  • no mapping experience
  • your data live in a geodatabase (and you don’t have ArcGIS Desktop)

f1u <- function(x) ifelse ((abs(x) >  3 & abs(x) <= 7), 3*sqrt(1-(x/7)^2), 0)
f1d <- function(x) ifelse ((abs(x) >= 4 & abs(x) <= 7), -3*sqrt(1-(x/7)^2), 0)
f2u <- function(x) ifelse ((abs(x) > 0.50 & abs(x) < 0.75), 3*abs(x)+0.75, 0)
f2d <- function(x) ifelse ((abs(x) > -4 & abs(x) < 4), abs(x/2)-(3*sqrt(33)-7)*x^2/112-3 + sqrt(1-(abs(abs(x)-2)-1)^2), 0)
f3u <- function(x) ifelse ((x > -0.5 & x < 0.5), 2.25, 0)
f4u <- function(x) ifelse ((abs(x) >  1 & abs(x) <= 3), 6 * sqrt(10)/7 + (1.5 - 0.5 * abs(x)) * sqrt(abs(abs(x)-1)/(abs(x)-1)) - 6 * sqrt(10) * sqrt(4-(abs(x)-1)^2)/14, 0)
f5u <- function(x) ifelse ((abs(x) >= 0.75 & abs(x) <= 1), 9-8*abs(x), 0)
fu <- function (x) f1u(x) + f2u(x) + f3u(x) + f4u(x) + f5u(x)
fd <- function (x) f1d(x) + f2d(x)
bm <- function(r,x) ifelse(r%%2==0, fu(x), fd(x))
par(bg="#02233f", mar=c(0,0,0,0))
plot(x=seq(-7,7,0.065), y=Vectorize(bm)(1:216, seq(-7,7,0.065)), col="#999900", type="l", lwd=3)
